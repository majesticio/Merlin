# Merlin 

## Installation
- `pipenv shell`
- `pip install -r requirements.txt`

## Run Merlin 
- `ollama serve`
- `python transcriber.py`
- `python speaker.py`
- `python responder.py`
- `sudo pythom recorder.py`

## Use recorder
- `sudo python recorder.py`
- hold the `spacebar` to record
- press `ctrl + c` to exit
